== Description ==

Zapier Pro Extension for Auto Funnel Press v2.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/afp_zapier' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the AFP Settings->AFP License screen to activate your license before being able to configure AFPv2.
